# راهنمای کاربر گوگ (Goug)

## ۱. معرفی

### گوگ چیست؟
گوگ یک ابزار متن‌باز، رایگان و مدرن برای گفت‌وگو با مدل‌های پیشرفته هوش مصنوعی مانند Gemini و ChatGPT است. این برنامه کاملاً در مرورگر شما اجرا می‌شود و نیازی به نصب یا ثبت‌نام ندارد.

### چه مشکلی را حل می‌کند؟
بسیاری از کاربران دوست دارند از مدل‌های مختلف هوش مصنوعی استفاده کنند، اما هر کدام رابط کاربری و تاریخچه مجزای خود را دارند. گوگ با فراهم کردن یک **محیط یکپارچه**، به شما اجازه می‌دهد تا به راحتی بین ارائه‌دهندگان مختلف جابجا شوید و تمام گفتگوهای خود را در یک مکان مدیریت کنید.

### چرا از گوگ استفاده کنیم؟ (مزایا)
- **حریم خصوصی کامل**: تمام گفتگوها و کلیدهای API شما فقط در مرورگر خودتان ذخیره می‌شوند و به هیچ سروری ارسال نمی‌گردند.
- **بدون نیاز به سرور**: گوگ یک برنامه کاملاً سمت کاربر (Client-Side) است. این یعنی می‌توانید آن را به صورت آفلاین اجرا کرده یا روی هر هاست ساده‌ای میزبانی کنید.
- **پشتیبانی از چند ارائه‌دهنده**: به راحتی بین مدل‌های Gemini, OpenAI و حتی مدل‌های محلی (مانند Ollama) جابجا شوید.
- **رابط کاربری سریع و زیبا**: طراحی مدرن و واکنش‌گرا که هم در دسکتاپ و هم در موبایل به خوبی کار می‌کند.
- **رایگان و متن‌باز**: شما می‌توانید کد منبع را مشاهده کرده و در توسعه آن مشارکت کنید.


## ۲. شروع سریع (۵ دقیقه)

1.  **دریافت کلید API**: از یکی از ارائه‌دهندگان هوش مصنوعی (مانند Google AI Studio) یک کلید API دریافت کنید.
2.  **اجرای گوگ**: برنامه را در مرورگر خود باز کنید. پنجره تنظیمات به صورت خودکار نمایش داده می‌شود.
3.  **وارد کردن اطلاعات**: کلید API و نام مدل مورد نظر خود را در بخش مربوطه وارد کرده و روی «ذخیره تغییرات» کلیک کنید.
4.  **شروع گفتگو**: پنجره تنظیمات بسته می‌شود. حالا می‌توانید اولین پیام خود را تایپ کرده و ارسال کنید!


## ۳. تنظیمات اولیه

برای استفاده از گوگ، به یک **کلید API** از ارائه‌دهنده هوش مصنوعی مورد نظر خود نیاز دارید. این کلید مانند یک رمز عبور است که به گوگ اجازه می‌دهد از طرف شما با مدل هوش مصنوعی صحبت کند.

#### دریافت کلید API

-   **برای Google Gemini**: به [Google AI Studio](https://aistudio.google.com/app/apikey) مراجعه کرده و یک کلید جدید بسازید.
-   **برای OpenAI (ChatGPT)**: به [داشبورد OpenAI](https://platform.openai.com/api-keys) مراجعه کرده و یک کلید جدید ایجاد کنید.

پس از دریافت کلید:
1.  در گوگ، روی دکمه **«مدیریت کلید API»** کلیک کنید.
2.  کلید خود را در فیلد مربوط به ارائه‌دهنده (مثلاً Gemini) وارد کنید.
3.  نام مدل را مشخص کنید (مثلاً `gemini-2.5-flash`).
4.  روی **«ذخیره تغییرات»** کلیک کنید.


## ۴. کار با گپ‌ها (Chats)

گوگ تمام گفتگوهای شما را به صورت خودکار ذخیره می‌کند.

-   **ایجاد گپ جدید**: روی دکمه **«+ گپ جدید»** در پایین پنل کناری کلیک کنید.
-   **جابجایی بین گپ‌ها**: روی عنوان هر گپ در پنل کناری کلیک کنید تا آن را باز کنید.
-   **تغییر نام گپ**: موس را روی یک گپ نگه دارید، روی دکمه سه‌نقطه کلیک کرده و گزینه «تغییر نام» را انتخاب کنید.
-   **حذف گپ**: از همان منوی سه‌نقطه، گزینه «حذف» را انتخاب کنید. **توجه:** این عمل قابل بازگشت نیست.


## ۵. ارسال و دریافت پیام

-   **نوشتن پیام**: پیام خود را در کادر متنی در پایین صفحه تایپ کنید.
-   **ارسال پیام**: کلید `Enter` را فشار دهید.
-   **ایجاد خط جدید**: برای رفتن به خط بعدی بدون ارسال پیام، `Shift+Enter` را فشار دهید.
-   **مشاهده پاسخ**: پاسخ مدل به صورت کلمه به کلمه (استریم) نمایش داده می‌شود تا حس یک گفتگوی زنده را داشته باشید.
-   **نشانگر تایپ**: زمانی که مدل در حال آماده‌سازی پاسخ است، یک انیمیشن تایپ نمایش داده می‌شود.


## ۶. ذخیره‌سازی و حریم خصوصی

**حریم خصوصی شما اولویت اصلی گوگ است.**

-   **ذخیره‌سازی محلی**: تمام داده‌ها، شامل تاریخچه گفتگوها و کلید API، در پایگاه داده داخلی مرورگر شما (IndexedDB) ذخیره می‌شوند. این داده‌ها هرگز از دستگاه شما خارج نمی‌شوند (مگر برای ارسال مستقیم به API ارائه‌دهنده انتخابی شما).
-   **بدون جمع‌آوری داده**: گوگ هیچ‌گونه اطلاعاتی از کاربران جمع‌آوری نمی‌کند.

## ۸. رفع مشکلات رایج (Troubleshooting)

-   **مشکل: پیام خطای "تنظیمات API صحیح نیست" را دریافت می‌کنم.**
    -   **راه‌حل**: مطمئن شوید که کلید API خود را به درستی در تنظیمات وارد کرده‌اید. کلید را دوباره کپی و جای‌گذاری کنید. همچنین، نام مدل را بررسی کنید که صحیح باشد.

-   **مشکل: پیام ارسال می‌شود اما پاسخی دریافت نمی‌شود.**
    -   **راه‌حل ۱**: اتصال اینترنت خود را بررسی کنید.
    -   **راه‌حل ۲**: ممکن است کلید API شما منقضی شده یا اعتبار کافی نداشته باشد. به وب‌سایت ارائه‌دهنده (مثلاً گوگل) مراجعه کرده و وضعیت حساب خود را بررسی کنید.
    -   **راه‌حل ۳**: ممکن است API ارائه‌دهنده به طور موقت با مشکل مواجه باشد. کمی صبر کرده و دوباره تلاش کنید.

-   **مشکل: تمام گپ‌های من پاک شده‌اند!**
    -   **راه‌حل**: گوگ داده‌ها را در حافظه مرورگر ذخیره می‌کند. اگر حافظه کش (Cache) مرورگر خود را پاک کرده باشید یا از حالت `Incognito/Private` استفاده کنید، ممکن است داده‌ها حذف شوند. متأسفانه، بازیابی آن‌ها ممکن نیست.

-   **مشکل: برنامه بارگذاری نمی‌شود یا صفحه سفید است.**
    -   **راه‌حل**: ابتدا صفحه را رفرش کنید (`Ctrl+R` یا `Cmd+R`). اگر مشکل ادامه داشت، حافظه کش مرورگر خود را برای این سایت پاک کرده و دوباره امتحان کنید.


## ۹. پرسش‌‌های پرتکرار (FAQ)

-   **آیا گوگ رایگان است؟**
    بله، استفاده از خود نرم‌افزار گوگ کاملاً رایگان است. اما توجه داشته باشید که استفاده از API مدل‌های هوش مصنوعی (مانند Gemini یا OpenAI) ممکن است طبق تعرفه‌های آن شرکت‌ها هزینه داشته باشد.

-   **آیا داده‌های من امن است؟**
    بله. همانطور که گفته شد، داده‌های شما فقط در مرورگر خودتان ذخیره می‌شوند. با این حال، به خاطر داشته باشید که ذخیره‌سازی کلید API در مرورگر برای پروژه‌های حساس توصیه نمی‌شود. گوگ برای استفاده شخصی و توسعه طراحی شده است.

-   **آیا می‌توانم در موبایل از گوگ استفاده کنم؟**
    بله، رابط کاربری گوگ کاملاً واکنش‌گرا (Responsive) است و به خوبی روی دستگاه‌های موبایل و تبلت کار می‌کند.

-   **چگونه می‌توانم از گپ‌ها نسخه پشتیبان تهیه کنم؟**
    در حال حاضر، گوگ گزینه داخلی برای پشتیبان‌گیری ندارد. از آنجایی که داده‌ها در پایگاه داده داخلی مرورگر (IndexedDB) ذخیره می‌شوند، استخراج آن‌ها برای کاربران غیرفنی به سادگی امکان‌پذیر نیست. افزودن قابلیت Export/Import یکی از برنامه‌های آینده برای توسعه این ابزار است.

-   **آیا می‌توانم چند کلید API را همزمان ذخیره کنم؟**
    خیر، در هر زمان فقط یک مجموعه تنظیمات (شامل یک ارائه‌دهنده، یک مدل و یک کلید) می‌تواند فعال باشد. برای استفاده از ارائه‌دهنده دیگر، باید تنظیمات را تغییر دهید.
