کلید API شما فقط در مرورگر شما ذخیره می‌شود. ما هیچ داده‌ای را جمع‌آوری نمی‌کنیم.

  • از کلیدهای API در دستگاه‌های شخصی و مورد اعتماد استفاده کنید.
  • برای کلیدهای خود محدودیت‌های استفاده (Usage Limits) تنظیم کنید.
  • کلیدهای خود را به صورت عمومی به اشتراک نگذارید.
